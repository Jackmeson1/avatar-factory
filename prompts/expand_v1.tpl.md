# AI Prompt模板：情节扩展

**模板版本**: v1.0
**适用场景**: 扩充剧情段落、补充细节
**最后更新**: 2024-12-XX

## 基础Prompt结构

```
你是一位小说创作助手，负责将简短的情节扩展成更加生动丰富的段落，同时保持原有语气和设定。
```

## 使用示例

### 示例1：扩写片段
```
## 当前文本
主角进入森林。

## 具体指导
- 加入环境描写
- 体现主角的紧张感

## AI输出预期
[AI扩写内容]
```

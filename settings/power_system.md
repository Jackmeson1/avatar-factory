# 能力系统概览

**最后更新**: 2025-06-07  
**版本**: v0.1  
**状态**: 草案

## 1. 能力阶梯

- **Tier 0 — 普通人**：无任何特殊力量。
- **Tier 1 — 觉醒者**：初步操控，效果有限。
- **Tier 2 — 精通者**：可在小范围内稳定发挥。
- **Tier 3 — 城市级**：具备影响大规模战局的能力。
- **Tier 4 — 灾难级**：足以摧毁国家或改变世界格局。
- **Tier 5 — 神话级**：触及宇宙或多维法则。

## 2. 使用代价公式

- **能量消耗**：`基础消耗 × (1 + 名气值 / 1000)`
  - *基础消耗* 随技能等级（1–5）每级 +10。
- **人格稀释**：`名气增长百分比 × 0.05`
  - 每新增 100 点名气，主角人格稀释 5‰。

## 3. 其他机制

- **冷却时间**：依能力等级从数秒至数日不等。
- **协同效果**：多化身协同时，消耗等比提升。
- **复苏成本**：化身被击败后可重组，但需额外名气。
